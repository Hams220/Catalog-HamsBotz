Deskripsi

Anda akan berlangganan VIP1 selama 7 hari (tidak termasuk sabtu minggu), anda bisa cek keuntungan langganan dibawah ini :

  • Notif ON Group
  • Notif ON Pribadi
  • Auto Claim
  • Free kode hydro

Mekanisme langganan :

User A berlangganan HamsBotz di Hari senin, makan langganan akan berakhir di hari selasa (minggu depan)

Mekanisme perubahan nomor :

User A berlangganan HamsBotz di hari senin, user A hanya dapat menyetorkan 1 nomor whatsap ke admin, lalu hari selasa nomor whatsap itu limit, maka admin akan menghubungi user A agar user A dapat menyetorkan kembali whatsapp barunya

FAQ

A: Tidak!, Whatsap anda tidak digunakan untuk spam, whatsap hanya digunakan untuk proses klaim

Q: Bagaimana jika whatsap saya terkena spam?
A: Admin tidak bertanggung jawab, kami sarankan agar anda membeli kartu perdaba agar bisa login ulang

Q: Apakah saya bisa melakukan pengembalian dana, jika berhenti langganan sebelum masa langganan saya habis?
A: Anda bisa melakukan pengembalian dana maximal 2 hari, setelah melakukan pembayaran (syarat & ketentuan berlaku)
